## Translating

Use [Weblate](https://hosted.weblate.org/projects/renewed-tab/website/)

### Updating translations

1. Copy `en` to `LOCALE`.
2. Delete any `redirect_from`.
3. Add your language to `_data/languages.yml`.
4. Translate any markdown (.md) files in `LOCALE`.
